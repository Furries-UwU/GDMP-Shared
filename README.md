# GDMP-Shared

Shared library used between the Client and Server

# Build Instruction

## MacOS

```sh
cmake . -DCMAKE_OSX_ARCHITECTURES=x86_64 -DGEODE_TARGET_PLATFORM=MacOS
make
```

## Windows

```cmd
cmake .
make
```
